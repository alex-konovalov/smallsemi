  [1X5.1 Properties of Semigroups[0X
  
  In  this  section  we detail the [5XGAP[0m functions that can be used to determine
  whether  a  small  semigroup  satisfies  a  certain  property.  Let  [3XS[0m  be a
  semigroup. Then
  
  --    [3XS[0m is a [13Xleft zero semigroup[0m if [3Xxy=x[0m for all [3Xx,y[0m in [3XS[0m.
  
  --    [3XS[0m is a [13Xright zero semigroup[0m if [3Xxy=y[0m for all [3Xx,y[0m in [3XS[0m.
  
  --    [3XS[0m is [13Xcommutative[0m if [3Xxy=yx[0m for all [3Xx,y[0m in [3XS[0m.
  
  --    [3XS[0m is [13Xsimple[0m if it has no proper two-sided ideals.
  
  --    [3XS[0m is [13Xregular[0m if for all [3Xx[0m in [3XS[0m there exists [3Xy[0m in [3XS[0m such that [3Xxyx=x[0m.
  
  --    [3XS[0m is [13Xcompletely regular[0m if every element of [3XS[0m lies in a subgroup.
  
  --    [3XS[0m  is  an  [13Xinverse  semigroup[0m  if  every  element  [3Xx[0m in [3XS[0m has a unique
        semigroup  inverse,  that  is,  a unique element [3Xy[0m such that [3Xxyx=x[0m and
        [3Xyxy=y[0m.
  
  --    [3XS[0m  is  a  [13XClifford  semigroup[0m  if  it  is  a  regular  semigroup whose
        idempotents  are  central,  that  is,  for all [3Xe,f[0m in [3XS[0m with [3Xe^2=e[0m and
        [3Xf^2=f[0m we have that [3Xef=fe[0m.
  
  --    [3XS[0m  is a [13Xband[0m if every element is an idempotent, that is, [3Xx^2=x[0m for all
        [3Xx[0m in [3XS[0m.
  
  --    [3XS[0m  is  a [13Xrectangular band[0m if for all [3Xx,y,z[0m in [3XS[0m we have that [3Xx^2=x[0m and
        [3Xxyz=xz[0m.
  
  --    [3XS[0m  is  a  [13Xsemiband[0m if it is generated by its idempotent elements, that
        is, elements satisfying [3Xx^2=x[0m.
  
  --    [3XS[0m  is  an  [13Xorthodox  semigroup[0m if its idempotents (elements satisfying
        [3Xx^2=x[0m) form a subsemigroup.
  
  --    [3XS[0m is a [13Xzero semigroup[0m if there exists an element [3X0[0m in [3XS[0m such that [3Xxy=0[0m
        for all [3Xx,y[0m in [3XS[0m.
  
  --    [3XS[0m  is  a  [13Xzero  group[0m  if  there  exists an element [3X0[0m in [3XS[0m such that [3XS[0m
        without [3X0[0m is a group and for all [3Xx[0m in [3XS[0m we have that [3Xx0=0x=0[0m.
  
  The  [5XMONOID[0m  was  used to determined which semigroups in the library satisfy
  the  properties  above.  All  of  the resulting information is stored in the
  library.

  [1X5.1-3 Is3NilpotentSemigroup[0m
  
  [2X> Is3NilpotentSemigroup( [0X[3Xsgrp[0X[2X ) ____________________________________[0Xproperty
  
  returns [10Xtrue[0m if the small semigroup [3Xsgrp[0m is 3-nilpotent and [10Xfalse[0m otherwise.
  
  A  semigroup  is [13X3-nilpotent[0m if it has a zero element and every product of [3X3[0m

    [4Xgap> Is3NilpotentSemigroup(s);[0X
    [4Xfalse[0X
    [4Xgap> s:=SmallSemigroup(8, 11433105+1231);;[0X
    [4Xgap> Is3NilpotentSemigroup(s);[0X
    [4Xtrue[0X
    [4Xgap> s:=SmallSemigroup(8,1);;[0X
    [4Xgap> Is3NilpotentSemigroup(s);[0X
    [4Xfalse[0X
    [4X	[0X
  [4X------------------------------------------------------------------[0X
  
  Note  that all the semigroups in the library of size [3X8[0m and id number greater
  than [3X11433105[0m are [3X3[0m-nilpotent.
