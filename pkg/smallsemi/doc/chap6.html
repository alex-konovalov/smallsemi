6.1 Finding all semigroups

PREAMBLE The following functions have the same usage but, of course, return different values: EnumeratorOfSmallSemigroups (6.1-2), AllSmallSemigroups (6.1-1), EnumeratorSortedOfSmallSemigroups (6.1-3), IdsOfAllSmallSemigroups (6.1-4), IteratorOfSmallSemigroups (6.1-7), NrSmallSemigroups (6.1-8), OneSmallSemigroup (6.1-9), PositionsOfSmallSemigroups (6.1-10), RandomSmallSemigroup (6.1-15). The number of arguments that these functions accept is any odd number: the first should be a positive integer, a list of positive integers, or an enumerator or iterator of small semigroups, the 2ith argument should be a function, and the 2i+1th argument should be a possible value that can be returned by the preceding function. For example, to obtain all the commutative semigroups with 3 idempotents of sizes 2 to 5 in a list we use:

AllSmallSemigroups([2..5], IsCommutative, true, Is3IdempotentGenerated, true);

or if we want a random semigroup with this property we use:

RandomSmallSemigroup([2..5], IsCommutative, true, Is3IdempotentGenerated, true);

6.1-10 PositionsOfSmallSemigroups



> PositionsOfSmallSemigroups( arg )( function )


the number of argument of this function should be odd. The first argument arg[1] should be a positive integer or an enumerator with IsEnumeratorOfSmallSemigroups (6.1-5), the even arguments arg[2i], if present, should be functions, and the odd arguments arg[2i+1] should be a value that the preceeding function can have. For example, a typical input might be 3, IsRegularSemigroup, true. The functions arg[2i] can be user defined or existing GAP functions. The argument can be a list arg with the same components as given above.



The function returns the second component of the IdSmallSemigroup (4.1-7) of all the small semigroups S in the library satisfying arg[2i](S)=arg[2i+1] for all i. For example, if the input is 3, IsRegularSemigroup, true, then the output is a list of the ids of all semigroups in the library of size 3 that are regular.






gap>  PositionsOfSmallSemigroups([3]);
[ 1 .. 18 ]
gap>PositionsOfSmallSemigroups(3);
[ 1 .. 18 ]
gap> PositionsOfSmallSemigroups(3, IsRegularSemigroup, false);
[ 1, 2, 3, 4, 5, 6, 7, 8, 9 ]
gap> enum:=EnumeratorOfSmallSemigroups(3, IsRegularSemigroup, false);
<enumerator of semigroups of size 3>
gap> PositionsOfSmallSemigroups(enum);
[ 1, 2, 3, 4, 5, 6, 7, 8, 9 ]
gap> PositionsOfSmallSemigroups(4, IsCommutative, true,                        
> IsRegularSemigroup, false, IsMonoidAsSemigroup, true);
[ 24, 28, 32, 40, 48, 68, 78, 83 ]

6.1-12 PrecomputedSmallSemisInfo



> PrecomputedSmallSemisInfo( global variable )


the global variable PrecomputedSmallSemisInfo contains a list of all the names of precomputed properties stored in the library. The ith element of the list contains the list of properties that have been precomputed for all semigroups in the library of order i.

6.1-17 UpToIsomorphism



> UpToIsomorphism( sgrps )( operation )


takes a list sgrps of non-equivalent semigroups from the library as input and returns a list of non-isomorphic semigroups containing for every semigroup in sgrps an isomorphic semigroup and an anti-isomorphic semigroup.
