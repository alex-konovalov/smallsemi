  [1X1.1-8 3NIL_DATA[0m
  
  [2X> 3NIL_DATA__________________________________________________[0Xglobal variable
  
  is a record carrying data to restore 3-nilpotent semigroups of size 8.
  
  At the time [5Xsmallsemi[0m is loaded only the component [10Xdiag[0m is bound and has the
  value  [10Xfail[0m.  This changes when a 3-nilpotent semigroup of size 8 is called.
  Then  [10Xdiag[0m becomes a list element from the component [10X3nildiags[0m of the global
  variable  [2XMOREDATA2TO8[0m  ([14X1.1-12[0m)  corresponding  to the diagonal of the last
  called 3-nilpotent semigroup of size 8.
  
  The other components are [10Xstrlist[0m, a list of strings carrying the information
  about  the entries of the stored multiplication tables; [10Xpositions[0m, a list of
  integers,  the  positions  of the stored solutions relative to the first one
  with  the same diagonal and [10Xnext[0m, an integer storing which position was last
  called for.
  
  [4X---------------------------  Example  ----------------------------[0X
    [4Xgap> LoadPackage("smallsemi");[0X
    [4Xtrue[0X
    [4Xgap> 3NIL_DATA;[0X
    [4Xrec( diag := fail )[0X
    [4Xgap> SmallSemigroup( 8, NrSmallSemigroups(8)-2 );;[0X
    [4Xgap> 3NIL_DATA;                                  [0X
    [4Xrec( diag := [ 2, 3 ], strlist := [ "0013", "0313" ], [0X
    [4X  positions := [ 1, 3, 4, 7 ], next := 4 )[0X
